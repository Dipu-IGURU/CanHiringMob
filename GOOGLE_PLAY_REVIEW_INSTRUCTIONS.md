### App Features Available After Login

- Browse jobs
- Search for jobs by title and location
- View job details
- Save jobs (bookmark functionality)
- Apply for jobs
- View profile
- Access dashboard

### Notes for Reviewers

- The app requires internet connection to function
- Some features may require backend server to be running
- Job listings are fetched from the server
- User can browse jobs without full registration, but login is required for most features

### Troubleshooting

If login fails:
1. Check internet connection
2. Verify backend server is running
3. Try creating a new account using the Sign Up option
4. Contact support if issues persist
